# Aurora Digital 模拟知识库目录

更新时间：2026-06-04

本知识库用于智能客服演示，内容为真实电商风格的仿真数据，不代表真实品牌承诺。

## 构建规则

- 一卡一主题，避免把产品、政策、售后混在同一文件。
- 使用 broad category：product、policy、promotion、after_sales、logistics、billing、account、compatibility、service、order。
- 每张卡包含 title、category、keywords、updated_at、priority，产品相关卡使用 product_ids。
- 正文包含具体参数、适用条件、例外、所需用户信息和升级人工条件。
- 不保存真实个人信息、真实订单、真实手机号、真实地址或真实支付流水。
